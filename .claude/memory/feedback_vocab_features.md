User wants comprehensive vocabulary list features:
- View word list
- Delete words
- Search/filter
- Mark words (e.g., mastered/needs review)
- Export to Anki-compatible format

**Why:** Complete workflow from capture to review - the export to Anki is critical for the user's spaced repetition study workflow.

**How to apply:** Design the vocabulary list UI with all these management features. For export, research Anki's import format (likely CSV or tab-delimited with specific field order) and generate files that can be directly imported into Anki decks.
