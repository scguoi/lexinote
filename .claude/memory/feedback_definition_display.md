---
name: Definition display preference
description: User prefers in-place tooltip card for showing word definitions
type: feedback
---

User wants word definitions to appear as a small card (tooltip-style) at the original position where the word was selected, rather than in a sidebar, drawer, or popup window.

**Why:** Keeps the reading flow uninterrupted and provides immediate context without navigating away from the reading position.

**How to apply:** Design the definition popup as a positioned overlay card that appears near the selected word, with minimal UI chrome to maintain the clean aesthetic.
