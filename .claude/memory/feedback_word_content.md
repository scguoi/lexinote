name: Word definition content requirements
description: User wants comprehensive word information including definition, phonetics, part of speech, example sentences, and etymology analysis
type: feedback
---

When looking up a word, the user wants the LLM to return:
- Definition (Chinese translation)
- Phonetics (pronunciation)
- Part of speech
- Example sentences
- Etymology analysis (word roots and affixes)

**Why:** Comprehensive information helps with deeper understanding and memory retention of vocabulary.

**How to apply:** Design the LLM prompt to request all these elements in a structured format. The UI card should display all these fields in a clean, scannable layout.
