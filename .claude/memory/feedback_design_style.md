User emphasizes the UI design should be:
- Modern (现代)
- Minimalist (简约)
- Beautiful (漂亮)
- Cute content elements (内容上可爱一点)
- Warm and friendly feeling (给人一些温度)

**Why:** The user wants the extension to feel approachable and pleasant to use, not cold or purely functional.

**How to apply:**
- Use soft color palette (pastels, warm tones)
- Add subtle animations and micro-interactions
- Consider friendly iconography (rounded, not sharp)
- Use warm, encouraging copy (e.g., "太棒了！已加入生词本 ✨" instead of just "已添加")
- Add delightful details like gentle hover effects, smooth transitions
- Consider emoji or illustration accents where appropriate
